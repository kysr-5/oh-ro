# MuJoCo 家居场景说明文档 (v3.1 - 优化布局版)

## 场景概述

这是一个适用于 MuJoCo 3.4.0 的仿真家居场景,包含**七个房间**:客厅、厨房、主卧、书房、主浴室、客浴和储藏室。场景采用真实比例设计,空间宽敞,所有物体位置符合日常生活逻辑,光照柔和舒适。

### 连通性设计
- **客厅 ↔ 厨房**: 客厅左墙在 Y=-0.5 至 Y=0.5 位置有 1 米宽门洞
- **客厅 ↔ 主卧**: 客厅右墙在 Y=-0.5 至 Y=0.5 位置有 1 米宽门洞
- **厨房 ↔ 书房**: 厨房前墙在 X=-13 至 X=-12 位置有 1 米宽门洞
- **书房 ↔ 主浴室**: 书房后墙在 X=-13.5 至 X=-12.5 位置有 1 米宽门洞
- **主浴室 ↔ 储藏室**: 主浴室后墙在 X=-14.5 至 X=-13.5 位置有 1 米宽门洞
- **主卧 ↔ 客浴**: 主卧前墙在 X=11 至 X=12 位置有 1 米宽门洞
- **自由通行**: 可从任意房间自由移动到其他房间,无障碍物阻挡

## 整体布局

```
    厨房 (10×8m) ←门→ 客厅 (12×10m) ←门→ 主卧 (8×8m)
         ↓门                              ↓门
    书房 (6×5m)                      客浴 (4×3m)
         ↓门
    主浴室 (4×3m)
         ↓门
    储藏室 (4×3m)
```

## 坐标系统

- **X 轴**: 左右方向 (厨房在负方向 X=-16 至 -6,主卧在正方向 X=6 至 14)
- **Y 轴**: 前后方向 (客厅前墙 Y=-5,客厅后墙 Y=5)
- **Z 轴**: 垂直方向 (地面为 Z=0,天花板约 Z=2.4)
- **原点**: 位于客厅中心

## 物理属性设置

### 材质系统
- **wood** (木质): 棕色 (0.6, 0.4, 0.2),用于桌子、椅子等
- **darkwood** (深色木质): 深棕色 (0.4, 0.2, 0.1),用于电视柜、床架等
- **fabric** (布艺): 蓝灰色 (0.3, 0.3, 0.5),用于沙发、床垫等
- **metal** (金属): 银灰色 (0.7, 0.7, 0.75),用于冰箱、台面等
- **ceramic** (陶瓷): 浅色 (0.9, 0.9, 0.95),用于花瓶、花盆等
- **plastic** (塑料): 深灰色 (0.2, 0.2, 0.2),用于垃圾桶、电视等
- **wall** (墙面): 浅色 (0.95, 0.95, 0.9),所有墙壁
- **floor** (地板): 浅棕色 (0.8, 0.8, 0.7),地面
- **glass** (玻璃): 浅蓝色 (0.8, 0.9, 1.0, 0.5),半透明
- **door_frame** (门框): 深棕色 (0.5, 0.35, 0.2),所有门框

### 碰撞设置
- 所有家具和墙壁均启用碰撞检测 (`contype="1" conaffinity="1"`)
- 装饰性元素(地毯)禁用碰撞 (`contype="0" conaffinity="0"`)
- 重力加速度: -9.81 m/s²

### 照明系统 (柔和设置)
- **主光源**: 顶部漫射光 (0, 0, 6),强度 0.3
- **辅助光源1**: 右上方 (8, 8, 4),强度 0.2
- **辅助光源2**: 左下方 (-8, -8, 4),强度 0.2
- **辅助光源3**: 右下方 (8, -8, 4),强度 0.2
- **辅助光源4**: 左上方 (-8, 8, 4),强度 0.2
- **头灯**: 环境光 0.3,漫射光 0.4,镜面反射 0.05

总体光照强度降低约 60%,营造温馨舒适的室内氛围。

## 使用说明

### 1. 加载场景
在 MuJoCo 3.4.0 中运行:
```bash
python -m mujoco.viewer --mjcf=home_scene.xml
```

或在 Python 代码中:
```python
import mujoco
import mujoco.viewer

model = mujoco.MjModel.from_xml_path("home_scene.xml")
data = mujoco.MjData(model)
mujoco.viewer.launch(model, data)
```

### 2. 推荐视角
- **全景视角**: 位置 (0, -10, 8),朝向 (0, 0, 0)
- **客厅视角**: 位置 (0, 0, 1.6),朝向电视
- **厨房视角**: 位置 (-11, 0, 1.6)
- **主卧视角**: 位置 (10, 0, 1.6)
- **书房视角**: 位置 (-13, -6.5, 1.6)
- **俯视图**: 位置 (0, -6, 15),朝向 (0, -6, -1)
